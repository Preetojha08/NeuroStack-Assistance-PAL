About NeuroAssist

PAL

NeuroAssist is an AI-powered service by NeuroStack that transforms the way neurological disorders are diagnosed and treated. By analyzing patient data such as medical images, genetic information, and clinical notes, NeuroAssist provides healthcare professionals with actionable insights and personalized treatment recommendations.

Our goal is to improve diagnostic accuracy, streamline workflows, and enhance patient care, empowering clinicians to make informed decisions faster and more effectively.

Android Project

Gtu Marks Calculator

The GTU Marks Calculator is an Android app tailored for Gujarat Technology University students, enabling them to easily calculate SPI, CPI, CGPA, and percentage. With a compact size of just 1MB, the app is quick to download and doesn’t take up much space on your device.

Despite its small footprint, it offers a smooth and reliable experience across all Android versions, allowing students to manually input their grades and get instant, accurate academic metrics. It has gained popularity with over 1,000+ downloads on the Play Store.

This project uses Java for development, Gradle and Maven for project management, Git and GitHub for version control, and OOP principles for a well-organized, reliable codebase.

Key Features:
♠ SPI, CPI, CGPA, and percentage calculation.
♡ User-friendly and responsive UI/UX design to ensure a seamless experience.
♢ Compact 1MB size for easy download.
♣ Smooth performance across all Android devices.

Android Project

COVID SofTest

COVID SofTest is an Android application designed to track global COVID-19 cases and provide essential information on symptoms and prevention methods. The app features a self-assessment test to help users evaluate their health status related to COVID-19.

The app provides a comprehensive tool for tracking the COVID-19 pandemic, offering valuable insights both globally and within India. It features a user-friendly interface designed to ensure an intuitive and accessible experience for all users.

The project is developed with Java, using Gradle for project management. It incorporates RESTful Services with PHP and JSON for data handling and utilizes threading for efficient background task management.

Key Features:
♠ Tracks global COVID-19 cases.
♡ Provides information on symptoms and prevention methods.
♢ Includes a self-assessment test for users.
♣ Intuitive design with seamless functionality.

Android Project

Todo List

To-Do Application is designed to help users efficiently manage their daily tasks and errands. It allows users to categorize tasks by priority (High, Medium, Low), add as many tasks as needed, and remove completed tasks with a simple click.

Developed with Java, utilizing Gradle for project management, RecyclerView for task list management, ViewModel and LiveData for handling UI-related data, Notification Manager for high-priority tasks, and Room Database for persistent data storage.

With a user-friendly and interactive interface, the app ensures an intuitive experience for all age groups, including students, married couples, and elderly individuals.

Key Features:
♠ Categorizes tasks by priority (High, Medium, Low) and allows for detailed descriptions.
♡ Adds unlimited tasks and removes completed ones easily.
♢ Intuitive and user-friendly interface for seamless interaction.
♣ Efficient management of UI data and notifications for high-priority tasks.
